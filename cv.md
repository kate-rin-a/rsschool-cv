[Codewars](https://www.codewars.com/kata/572fdeb4380bb703fc00002c) : coding in function accept 1 parameters arr, it's a string array. Your task is to put a character "|" into the middle of each element. For example: "abcd" should became "ab|cd"
```
    function isolateIt(arr){

  return arr.map((item) => {
  if (item.length % 2 === 0) {
    return item.slice(0, item.length/2) + '|' + item.slice(item.length/2);
  } else {
    return item.slice(0, item.length/2) + '|' + item.slice(item.length/2+1);
  }
  });
}
```
